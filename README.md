# ⚡ Teste Trovão – Programação Orientada a Objetos (Java)

Este repositório contém a resolução de um exercício proposto no chamado **“Teste Trovão”**, uma metodologia utilizada pelo professor como forma de **treinamento intensivo para provas**.

A ideia do Teste Trovão é simular, na prática, o tipo de problema que será cobrado na avaliação, permitindo que os alunos pratiquem conceitos fundamentais de forma aplicada.

---

## 🎯 Objetivo do Exercício

O objetivo principal desta atividade é praticar conceitos essenciais de **Programação Orientada a Objetos (POO) em Java**, como:

- Criação de classes  
- Encapsulamento (getters e setters)  
- Construtores  
- Uso de `ArrayList`  
- Sobrescrita do método `toString`  
- Manipulação de objetos  

---

## 📌 Descrição da Tarefa

### 1️⃣ Classe `Produto`

Criar uma classe chamada **Produto** contendo:

- Atributos (conforme o diagrama de classes fornecido)  
- Construtores  
- Métodos getters e setters  
- Método `toString`  

---

### 2️⃣ Classe `Loja`

Criar uma classe chamada **Loja** contendo:

- Atributos (incluindo um `ArrayList<Produto>`)  
- Construtores  
- Métodos getters e setters  
- Método `toString`  
- Métodos adicionais definidos no diagrama de classes  

A classe deve ser responsável por **armazenar e gerenciar os produtos disponíveis**.

---

### 3️⃣ Classe Principal (Execução)

Criar uma classe responsável por:

- Instanciar uma loja  
- Receber dados do usuário  
- Adicionar produtos à loja  
- Exibir:  
  - 🟢 Produto mais caro  
  - 🔵 Produto mais barato  
  - 🟡 Média de preços dos produtos
